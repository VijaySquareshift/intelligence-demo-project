# Intelligence Demo Project

This is a demo repository for testing the n8n Intelligence Workflow.

## Features

- OAuth 2.0 Authentication
- Database Query Optimization
- API v2 Documentation
- Mobile App (iOS/Android)
- Payment Gateway Integration

## Tech Stack

- Backend: Node.js, Express
- Frontend: React, TypeScript
- Database: PostgreSQL
- Cache: Redis

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

## Testing

\`\`\`bash
npm test
